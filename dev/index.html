Cim.contr_int_ho — Method

contour integral method with high-order moments pbar: the number of the moments (for p = 0, ..., pbar)

In this function, we follow the notations in Stefan Guttel, Francoise Tisseur, Acta Numerica, 2017 r,l : size of the probing matrices
